## CLI Usage

```bash

# Run benchmark and optionally show detailed metrics
python -m breakthrough_engine baseline compare --verbose
```

## Regression Policy

A Phase 6 change is accepted only if:
- No metric regresses by more than 0.05 vs Phase 5 baseline
- Champion rollback is triggered automatically if regression detected
